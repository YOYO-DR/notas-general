video-5 Video 5: Clásico Hola Mundo

    Crear proyecto

  • Como va a ser el primer proyecto, abrimos el IDE y damos clic en New Project
  • Nos abrirá esta ventana y le damos un nombre al proyecto y la ruta donde se va a guardar
  • Y damos clic en Create
  • Nos cargará con el main y un código de ejemplo
  • Eliminamos ese archivo, dando clic derecho sobre el y dando clic en la opción Delete...

    • Crear paquete

  • En la carpeta src, damos lic derecho y damos clic en New y luego en Package
  • Le damos un nombre y le damos Enter
  • Luego creamos un Java Class en el paquete, dando clic derecho encima del paquete y luego New y después en Java Class
  • Le damos un nombre, le damos clic en Class y luego Enter

    • Archivo Java Class

  • El archivo se crea, al inicio aparece package nombre; el cual le indica a Java a que paquete pertenece
  • La clase generada, es la clase principal del archivo, la cual siempre es el mismo nombre del archivo
  • Dentro de esa clase, creamos la función principal para ejecutar el archivo, el cual es la función main
    • package HolaMundo;

      public class holaMundo {
          public static void main(String[] args) {
              
          }
      }
  • Más adelante se ira aprendiendo como funciona Java, por ahora, se debe saber que en la función main, dentro va a ir todo el código a ejecutar, y para nuestro Hola mundo, se va a realizar de la siguiente manera, dentro de la función main
    • System.out.println("Hola mundo");
  • Después de cada sentencia de código, no olvidar poner los ;
  • Para ejecutar nuestro archivo, le damos clic en el triangulo de la parte superior, la cual abrirá una terminal y ejecutará nuestro proyecto, verificamos que diga Current File
  • Y así, nuestro primer Hola Mundo en Java
