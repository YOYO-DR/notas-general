Sección 1: Crear proyecto de Reactjs con create-react-app

  • Ya teniendo instalado Nodejs y visual studio code, abrimos una carpeta en visual studio code, abrimos una terminal con ctrl + ñ y ejecutamos
    • npx create-react-app nombre-app
  • Descargara unas librerias necesarias y creará el proyecto
  • Entramos a la carpeta del proyecto desde la terminal con cd nombre-app y ahi ya ejecutamos el servidor de desarrollo
    • npm start

Sección 2: Primer Hola Mundo en Reactjs

    NOTA: No tocar la carpeta node_modules y el archivo package-lock.json que es información de Node

  • En el proyecto creado, se crea una carpeta src en donde van los archivos de nuestro proyecto la cual vamos a modificar, el principal es el index.js que es el que el servidor buscará para mostrar, en este caso, borraremos la carpeta y la volveremos a crear, ya vacia
  • Creamos un archivo llamado index.js dentro de src e importamos la libreria de React y la de ReactDOM, React es la libreria nativa de React, pero para manejar el FrontEnd, utilizamos el ReactDOM
    • import React from 'react'
import ReactDOM from 'react-dom/client'
  • Ya luego, creamos una constante para crear el root o donde va nuestra aplicación, porque si se revisa el index.html en la carpeta public hay un div con el id de root, la cual js selecciona ese div y ahi ira todo el proyecto de react
  • Creamos la contante, con ReactDOM y le pasamos a su funcion, el elemento root seleccionado con el dom de Js
    • const root = ReactDOM.createRoot(document.getElementById("root"));
  • Ya de esta forma podemos manipular ese div con sintaxis de react, en este caso, le paso un h1 con el hola mundo 
    • root.render(<h1>Hola mundo</h1>);
  • Y asi, con la función render le pasamos ese h1 dentro del root y si modificamos lo que le pasamos y guardamos, se actualiza automaticamente
